# RoboRoster

RoboRoster is a single-page web app is setuped with React + Vite, that cover the basic concepts of React.

## Features

- Card
- Card List
- Search Box

## Tech Stack

- **React Basics** 
- **JavaScript**
- **CSS**

## Project Setup

### Install Dependencies

```sh
pnpm install
```

### Run in Development Mode

```sh
pnpm dev
```

### Build for Production

```sh
pnpm build
```
